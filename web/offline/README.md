# Web Offline Demo

This demo shows a browser-based Web client connecting to a demo server.

## Setup

1. Start a server from `../../servers/`.
2. Open or serve this folder with the selected Web toolchain.
3. Configure the demo server base URL and assigned CIDI proxy API key.
4. Run the demo page and trigger server-backed actions.

## Notes

Keep server API behavior aligned with the server implementation used by this demo.
