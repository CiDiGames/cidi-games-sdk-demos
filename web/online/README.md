# Web Online Demo

This demo shows direct CIDI Game SDK integration from a browser-based Web client.

## Setup

1. Open or serve this folder with the selected Web toolchain.
2. Import the CIDI Game SDK according to the SDK documentation.
3. Apply values based on `../../shared/config-examples/online.example.json`.
4. Run the demo page and inspect browser console output.

## Notes

SDK source code and private credentials should not be committed.
