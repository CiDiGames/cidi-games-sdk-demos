# Changelog

All notable changes to this repository will be documented in this file.

## 0.1.0 - 2026-06-15

### Added

- Initialized the CIDI Game SDK demos repository structure.
- Added client demo folders for Cocos Creator 2.x, Cocos Creator 3.x, Unity, Godot, and Web.
- Added online and offline mode README files for each client platform.
- Added shared configuration examples, test account template, and protocol notes.
- Added server-side demo folders for Node.js, Python, Java, and Go.
- Added a NestJS Node.js server demo for CiDi OpenAPI integration.
- Added request signing helpers for `X-Api-Key`, `X-Timestamp`, `X-Nonce`, and `X-Signature`.
- Added demo endpoints for token verification, balance query, payment orders, payment records, medals, tournament scores, daily tasks, report query, and test coin grant.
- Added callback endpoints for signed order payment callbacks and tournament score result callbacks.
- Added official documentation link to the root README.
